Please follow below steps for setup Email_Classification_App:

Step 1: Need to install Nodejs         ( Node version: v12.18.3 and npm version: 6.14.6)  (Note: This npm will automaticaly be installed after installing Nodejs)
Step 2: Need to install Redis Database ( version: 3.0.504)
Step 3: Go to Path where index.js file exist in terminal
Step 4: run               - npm install
step 5: run               - node index.js


Setup for Email Account:
Step 1: Write required input fields value under .env file like Email details, Path of Python Main file, port number etc.
Step 2: Setup IMAP configuration under all Email Accounts
Step 3: Create same Lables under All Emals Acount which used in the Application
